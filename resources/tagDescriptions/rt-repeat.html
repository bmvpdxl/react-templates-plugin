rt-repeat
Repeats a DOM node with its subtree for each item in an array.
The syntax is rt-repeat="itemVar in arrayExpr", where the element, itemVar, will be available in JavaScript context, and an itemVarIndex will be created to represent the index of the item.
By using this naming scheme, repeated expressions have access to all levels of nesting.
Online documents

Sample:


    <div rt-repeat="myNum in this.getMyNumbers()">{myNumIndex}. {myNum}</div>

Compiled:

define([
    'react',
    'lodash'
], function (React, _) {
    'use strict';
    function repeatMyNum1(myNum, myNumIndex) {
        return React.DOM.div({}, myNumIndex + '. ' + myNum);
    }
    return function () {
        return _.map(this.getMyNumbers(), repeatMyNum1.bind(this));
    };
});
